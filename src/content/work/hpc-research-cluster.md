Architected and automated an AWS-based High Performance Computing (HPC) platform to support HERA & Artemis II research simulation workloads. Built event-driven orchestration and compute provisioning systems enabling reliable execution of distributed scientific simulations across dynamically provisioned EC2 infrastructure.

Highlights:
* Designed event-driven HPC workload orchestration using Python and AWS services, enabling automated provisioning and execution of distributed simulation jobs.
* Orchestrated dynamic scaling of up to 48 EC2 compute nodes for parallel simulation workloads with automated job lifecycle management and retry-based fault recovery.
* Optimized compute utilization through automated resource allocation and workload scheduling, improving execution efficiency and system reliability.
* Implemented centralized logging, monitoring, and observability pipelines to improve system visibility and reduce time-to-diagnosis for production issues.
* Delivered a repeatable, cloud-based research computing platform supporting large-scale simulation workflows for NASA Artemis II research initiatives.
