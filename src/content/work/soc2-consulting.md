Acted as the primary technical consultant to drive a client's infrastructure toward SOC2 compliance. I was responsible for translating high-level auditor requirements into actionable engineering tasks and deploying the necessary security controls across their cloud environment.

Integrated their Google Workspace with AWS IAM Identity Center for centralized SSO. Eliminated static keys and bastion hosts by enforcing secure, auditable terminal access to instances via AWS SSM Session Manager.

Deployed AWS WAF and CloudFront to protect external-facing applications. Implemented continuous vulnerability scanning and intelligent threat detection across the infrastructure using Amazon GuardDuty and Intruder.
